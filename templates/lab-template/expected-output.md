# Expected Output

The default demo should print output similar to:

```text
scenario=baseline
step=1 result=accepted
step=2 result=observed
summary=replace this with lab-specific evidence
```

Exact numbers may differ only when the lab documents controlled randomness,
time, or input parameters.

Use this file to explain:

- which lines prove the expected behavior;
- which values should change when parameters change;
- which warnings or failures are intentional in a failure-mode scenario.
